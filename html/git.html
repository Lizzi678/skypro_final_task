Git

  • git init — инициализирует локальный репозиторий в текущей папке
  • git clone [url] — копирует существующий удаленный репозиторий на ваш компьютер
  • git add — добавляет все измененные файлы в индекс для подготовки к коммиту
  • git commit -m "сообщение" — сохраняет изменения, создавая «снимок» (коммит) с описанием
  • git status — показывает состояние рабочего дерева (какие файлы изменены или подготовлены)
  • git push — отправляет ваши локальные коммиты в удаленный репозиторий
  • git pull — забирает последние изменения из удаленного репозитория и объединяет их с вашими
